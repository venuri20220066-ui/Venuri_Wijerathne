## Academic Stress Analysis using PCA, Clustering & Machine Learning

This repository contains my Final Year Research Project conducted as part of the BSc (Hons) Data Science and Analytics programme at the University of Westminster.

Project Overview

This study focuses on analysing **academic stress among undergraduate students** of the Faculty of Management & Finance, University of Colombo.

The research adopts a **data-driven approach** to:
- Identify underlying stress dimensions
- Segment students based on stress profiles
- Predict stress levels using machine learning

The dataset consists of **1,763 student responses** collected using an adapted PAS (Perception of Academic Stress) scale.

 Methodology

The project is divided into two main stages:

Study 1 – Exploratory & Unsupervised Analysis
- Data Cleaning & Preprocessing  
- Descriptive Analysis  
- Inferential Analysis (Welch’s t-test, ANOVA)  
- Principal Component Analysis (PCA)  
- Varimax Rotation  
- Clustering (K-Means & Hierarchical)  

Study 2 – Predictive Modelling
- Feature Engineering  
- Data Preparation  
- Multi-output Regression  
- Models Used: Linear Regression, Random Forest, XGBoost  
- Model Evaluation (R², RMSE, MAE)

Key Findings

- Academic stress is **multi-dimensional**, consisting of:
  - Workload & Exam Pressure  
  - Academic Stress & Future Concerns  
  - Self-Confidence  
  - Lecturer-Induced Pressure  

- PCA explained approximately **51.2% of total variance**

- Students were segmented into **3 clusters**:
  - High Stress  
  - Moderate Stress  
  - Low Stress  

- **Linear Regression outperformed complex models**, indicating relationships are largely linear
